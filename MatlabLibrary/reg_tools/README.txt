Regularization tools is a Matlab package for analysis and solution of discrete
ill-posed problems, developed by Prof. Per Christian Hansen, Dept. of
Mathematical Modelling, Technical Univ. of Denmark.

INSTALLATION
To install the package:

1. Download the package from Matlab Central:
http://www.mathworks.com/matlabcentral/fileexchange/loadFile.do?objectId=52&objectType=file

2. Extract the package into this directory, or another directory in your
Matlab path.

MORE INFORMATION
For more information about regtools, visit Prof. Per Christian Hansen's
website:
http://www2.imm.dtu.dk/~pch/Regutools/
